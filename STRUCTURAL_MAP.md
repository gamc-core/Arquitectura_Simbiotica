# SYSTEM STRUCTURE

Arquitectura Simbiótica defines the structural architecture under which hybrid systems may emerge.

Within this architecture, several structural layers define how experimental systems may be executed and recorded.

- REGIME defines the experimental protocol used to execute structural experiments.

- ENDO defines a generational infrastructure capable of recording irreversible structural emissions.

- ENDO NODE is a runtime implementation of the ENDO protocol.

- Artifacts represent concrete outputs produced through structural execution.

- Execution traces record observable structural events and integrate them into the evolving trajectory of the system.
